# Rules for Organizational Behavior Course Workspace

## Language Rules

- The course is taught in **English**; all primary content should be in English
- Provide **bilingual support** (English + 中文) for key terminology and difficult concepts
- Use academic English appropriate for undergraduate study
- When explaining new terms, format as: **English Term** (中文翻译) — definition

## Content Rules

- Always cite the relevant OB theory or framework by **theorist name + year** when answering questions
- Reference specific textbook chapters where applicable
- Provide real-world organizational examples (e.g., Google, Apple, Huawei) to illustrate abstract concepts
- When comparing theories, use tables for clarity
- Clearly distinguish between descriptive theories (what is) and prescriptive theories (what should be)

## Output Templates

### Theory Card Template

When explaining an OB theory, use this structure:

```
## [Theory Name] (中文名)
- **Theorist**: Name (Year)
- **Core Idea**: 1-2 sentence summary
- **Key Components**: Bullet list of main elements
- **Strengths**: Why this theory is valuable
- **Limitations**: Known criticisms or gaps
- **Real-World Example**: A concrete case
- **Exam Tip**: What to focus on for tests
```

### Vocabulary Card Template

When explaining OB terminology, use this structure:

```
## [English Term] (中文翻译)
- **Definition**: Academic definition
- **Example Sentence**: "..."
- **Related Theory**: Which OB theory/framework it belongs to
- **Contrast With**: Similar or opposite terms
```

### Case Analysis Template

When analyzing an organizational case, use this structure:

```
## Case: [Case Title]
1. **Background**: Brief context
2. **Problem Identification**: Core issue(s)
3. **Theory Application**: Which OB theories explain the situation
4. **Analysis**: Detailed application of theory to facts
5. **Recommendation**: Practical suggestions
6. **Key Takeaway**: One-sentence lesson
```

## File Organization Rules

- Store notes in markdown (`.md`) format
- Organize content by chapter or topic area
- Use consistent naming conventions: `Chapter_XX_Topic_Name.md`
- Keep vocabulary lists, case analyses, and study notes in separate folders
- Use numbered prefixes for sequential ordering (e.g., `01_`, `02_`)

## Cross-Course Rules

- When OB content touches on **communication theory**, connect to 语言学 (Linguistics) concepts
- When OB content involves **cultural analysis**, connect to 美国文学 cultural context
- When writing academic responses, apply 高英 (Advanced English) writing standards
- Cross-references should be brief; link rather than duplicate

## Quality Rules

- Ensure accuracy of OB concepts and definitions by cross-referencing the textbook
- Flag any areas of academic debate or controversy with `> [!NOTE]`
- Keep explanations concise but comprehensive
- Prefer evidence-based content over anecdotal claims
