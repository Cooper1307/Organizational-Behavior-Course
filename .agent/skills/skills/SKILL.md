## Core Models & Frameworks

### OB Model (Ch1)
```
Inputs → Processes → Outcomes
- Individual Level (Ch2-8)
- Group Level (Ch9-14)
- Organizational Level (Ch15-17)
```

### Attitude Components (Ch2)
- Cognitive + Affective + Behavioral

### EVLN Model (Ch2)
- Exit (Active-Destructive)
- Voice (Active-Constructive)
- Loyalty (Passive-Constructive)
- Neglect (Passive-Destructive)

### Big Five Model (Ch4)
- Openness, Conscientiousness, Extraversion, Agreeableness, Neuroticism (OCEAN)

### Attribution Theory (Ch5)
- Distinctiveness, Consensus, Consistency

### Motivation Theories (Ch7)
- Expectancy = Expectancy × Instrumentality × Valence
- Equity: My outcomes/My inputs vs. Referent's outcomes/Referent's inputs

### Job Characteristics Model (Ch8)
- MPS = [(SV + TI + TS) / 3] × Autonomy × Feedback
