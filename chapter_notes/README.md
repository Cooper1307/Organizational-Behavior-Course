# Chapter Notes (章节笔记)

存放按章节组织的组织行为学学习笔记。

## 命名规范

- 格式：`Chapter_XX_Topic_Name.md`
- 使用数字前缀排序（如 `01_`, `02_`）

## 目录结构

```
chapter_notes/
├── Ch01_Introduction_to_OB.md
├── Ch02_Attitudes.md
├── Ch04_Personality_Values.md
├── Ch05_Perception.md
├── Ch07_Motivation.md
├── Ch09_Communication.md
├── Ch12_Leadership.md
├── Ch15_Organizational_Structure.md
└── Ch16_Organizational_Culture.md
```

## 教材精读笔记

精读原版教材段落的分析笔记，按以下格式命名：
- `Ch{章号}_{主题关键词}_paragraph_analysis.md`
- 例如：`Ch07_Motivation_expectancy_theory_paragraph_analysis.md`
