# 组织行为学 Organizational Behavior

> **授课语言**：English（英语专业课）
> **课程层次**：本科大三
> **学期**：2025-2026 春季学期

## 目录结构

```text
组织行为学/
├── .agent/                          ← AI 助手配置
│   ├── skills/SKILL.md              ← 课程技能 & 理论映射
│   ├── rules/rules.md               ← 输出规则 & 模板
│   └── workflows/                   ← 学习工作流
│       ├── study_vocabulary.md      ← /study_vocabulary 术语学习
│       ├── analyze_case.md          ← /analyze_case 案例分析
│       └── exam_review.md           ← /exam_review 考试复习
├── vocab/                           ← 术语表（按章节）
├── cases/                           ← 案例分析
├── notes/                           ← 课堂笔记
├── review/                          ← 复习材料
├── 课程大纲.md                       ← 课程安排
├── README.md                        ← 本文件
└── .ignore                          ← 忽略规则
```

| 命令 | 功能 |
