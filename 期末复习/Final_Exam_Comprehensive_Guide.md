## 第八部分：易混淆概念对比 / Commonly Confused Concepts

| 概念A / Concept A | 概念B / Concept B | 区分关键 / Key Difference |
|-------------------|-------------------|--------------------------|
| 成功管理者 | 有效管理者 | 成功=晋升快（社交网络）；有效=绩效高（沟通） |
| 表面多元化 | 深层多元化 | 表面=易察觉（性别、年龄）；深层=价值观、人格 |
| 基本归因错误 | 自利偏差 | FAE=判断他人（低估外部）；SSB=判断自己（成功=内部） |
| 群体思维 | 群体偏移 | 思维=从众阻止批判；偏移=夸大立场向极端 |
| 工作群体 | 工作团队 | 群体=绩效=总和；团队=积极协同，绩效>总和 |
| 分配式谈判 | 整合式谈判 | 分配式=零和、固定馅饼；整合式=双赢、扩大馅饼 |
| 机械结构 | 有机结构 | 机械=僵化、严格控制；有机=灵活、跨职能 |
| 表层扮演 | 深层扮演 | 表层=隐藏感受；深层=改变内心感受 |
| 交易型领导 | 变革型领导 | 交易型=合规、现状；变革型=超越利益、四个I |
| 强制权力 | 法定权力 | 强制=恐惧基础；法定=职位基础 |
| 挑战压力源 | 阻碍压力源 | 挑战=工作量、时间（可成长）；阻碍=繁文缛节（阻碍目标） |
| 敏感性训练 | 过程咨询 | 敏感性=无结构群体互动；过程咨询=外部顾问协助 |
